DESCRIPTION : 
	*The code has functions to add,delete,search and modify academic record.The record can be entered manually or read from file.
	*All records are written in a file too.If record is deleted from list, it gets deleted from file too.If it is modified, then the 	modified data is written in file too.
	*Like academic record, the code has functions for personal data of student too.
	*It also has functions to add T1,T2,ESE marks of student for a particular semester and a function to find the total marks.
	*It has a function to display all the details of a student for a particular semester.
